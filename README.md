## 프로젝트 소개
> **개발 기간** : 2024. 09. 12 (목) ~ 2024. 10. 03 (목)
> **개발 인원** : 1인 개발
> **최소 버전** : iOS 16.0+
> 
<br />

## 사용 기술 및 개발 환경
- **iOS** : Swift 5, Xcode 15.3
- **UI** : SwiftUI, Swift Charts
- **Architecture** : MVVM (Input-Output)
- **Reactive** : Combine
- **Network** : URLSession (Swift Concurrency)
- **Local DB** : RealSwift
<br />

## 주요 기능
### Google Places API 및  MaKit을 활용한 위치 기반 맛집 추천
- CoreLocation과 MapKit을 통합하여 실시간 사용자 위치 추적 및 권한 관리 구현
- Publishers.CombineLatest를 활용해 위치 정보와 맛집 데이터를 실시간으로 결합하여 표시
### Realm DB를 활용한 리뷰 관리 시스템
- Repository Pattern을 적용하여 일관된 리뷰 CRUD 작업 
- Auto Refresh 기능으로 리뷰 데이터 변경사항 실시간 반영
### GeometryReader와 DragGesture를 결합한 커스텀 카드 뷰 
- 
### Swift Charts를 활용한 버전별 차트 
- iOS 버전 별 차트 분기처리
